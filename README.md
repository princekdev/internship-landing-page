# Internship Landing Page 🚀

A modern, responsive internship programs landing page designed to showcase multiple internship opportunities with a clean UI, smooth animations, and an immersive glassmorphism design.

The project focuses on creating a premium frontend experience using pure HTML and CSS, with emphasis on accessibility, responsiveness, and modern UI practices.

---

## 🌐 Live Demo

🔗 Live Website: https://internship-landing-page-kappa.vercel.app/

---

## 📌 Project Overview

This project is a frontend internship listing website that displays different technology internship programs such as:

- Web Development Internship
- App Development Internship
- Python Programming Internship
- Java Programming Internship

Each program card provides essential details including:

- Internship duration
- Work location
- Monthly stipend
- Starting date
- Application link

The goal of this project is to build an engaging and professional internship discovery interface suitable for students and organizations.
